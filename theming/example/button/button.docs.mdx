A Button component to show theming usage.

### Wrap the component in a ThemeProvider to apply the theme

Change the overrides property from `pinkTheme` to `darkTheme` or `lightTheme` to see the theme change or remove this property to see the default theme.

```js live
<Theme.ThemeProvider overrides={pinkTheme}>
  <div>
    <Button>Hello world</Button>
    <button
      style={{
        color: 'var(--my-color-text)',
        backgroundColor: 'var(--my-color-background)',
        border: '2px solid var(--my-border-color)',
        borderRadius: 'var(--my-border-radius)',
        marginLeft: '10px',
        padding: '5px',
        fontSize: 'var(--my-font-size)',
      }}
    >
      hello world!
    </button>
  </div>
</Theme.ThemeProvider>
```

### Override the default theme with a pink theme

```js live
<Theme.ThemeProvider overrides={pinkTheme}>
  <Button>Pink Theme</Button>
</Theme.ThemeProvider>
```

### Override the default theme with a pink theme

This works the same as the example above

```js live
<PinkTheme>
  <Button>Pink Theme</Button>
</PinkTheme>
```

### Render button with different themes

```js live
<Theme.ThemeProvider>
  <Button>Default Theme</Button>

  <PinkTheme>
    <Button style={{ marginTop: '10px' }}>Pink Theme</Button>
  </PinkTheme>
</Theme.ThemeProvider>
```
